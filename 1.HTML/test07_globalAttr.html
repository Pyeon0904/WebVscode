전역 속성

모든 HTML 요소에서 사용할 수 있는 속성

id 속성

요소에 고유한 식별 ID를 지정
문서 내 다른 요소와 중복될 수 없음
- 이름은 하나 이상의 문자가 포함되어 있어야하며, 공백문자가 없어야함
- HTML, CSS, Javascript에서 요소를 구분하기 위한 용도로 주로 사용함

class 속성

요소에 하나 이상의 이름을 지정
문서 내 여러 요소에 같은 이름 지정 가능
이름은 알파벳, 숫자, 하이픈, 밑줄로 구성(시작은 알파벳으로 지정해야함)

hidden 속성

요소가 아직 또는 더이상 관련이 없음을 지정(요소 숨김 지정)
다른 조건이 충족될 때까지 사용자가 요소를 보지 못하게 하는데 사용(네이버 접기기능)
Javascript 를 이용해 속성 변경이 가능하다!

title 속성

요소에 대한 추가 정보를 지정
요소 위로 마우스를 이동하면 추가 정보가 "툴팁 텍스트"로 표시됨(네이버로고에 마우스 갖다대면 뭐라고 나오는 것)

style 속성

요소에 대한 인라인 CSS 스타일 지정
여러 스타일을 지정할 경우 ;(세미콜론)으로 구분을 한다!
style 태그 or 외부 스타일 시트에 지정된 스타일보다 우선 적용됨
