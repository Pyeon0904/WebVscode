CSS 기본 선택자

선택자 : 특정한 HTML 태그를 선택할 때 사용하는 기능

모든 선택자

HTML 문서 안의 모든 태그를 선택

 * {color : red} 와 같은 형태로 사용됨

태그 선택자

HTML 문서 내에 같은 태그를 모두 선택

 p { color : blue; }와 같은 형태로 사용됨

아이디 선택자

HTML 문서 내에 해당하는 아이디 속성 값은 가진 태그만 선택

 #아이디명 { color : green; }와 같은 형태로 사용됨

클래스 선택자

HTML 문서 내에 여러 개의 태그를 동일한 클래스명으로 지정해 선택 가능

 .클래스명 { color : purple; } 와 같은 형태로 사용됨
